## 版式说明

- **第 1～2 页**：封面签署表、修订记录 — 完整保留《软件概要设计说明》表格版式
- **第 3 页起**：**一份合并目录**（`TOC 1/2` 样式，15.35cm 右对齐点线页码）
- **正文**：宋体**五号**（10.5pt）、**单倍行距**；四部分以段前分页衔接

## 重新生成

生成前请关闭已打开的 Word 文档。

```bash
cd backend
python scripts/generate_merged_doc.py
```

优先从 `工程文档模板/15－软件概要设计说明V1.0.doc` 复制第 1～2 页版式；若本机 Word 不可用，则使用 python-docx 备用方案。
